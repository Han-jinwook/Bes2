# 🚩 주말 작업 가이드 (Bes2 v1.2.6)

## ✅ 오늘 완료된 사항
1. **가족사진 완벽 보호:** 분류 전략을 '블랙리스트(쓰레기만 정의)'로 전환하여 인물 사진이 쓰레기통으로 가는 문제 해결.
2. **앱 공유 기능:** 홈 화면 로고 클릭 시 깔끔한 앱 소개 + 설치 링크 공유 기능 추가.
3. **무한 로딩 해결:** 스크린샷 삭제 후 UI가 멈추던 뷰모델 로직 수정 완료.
4. **시스템 안정화:** 안드로이드 14 타임아웃 크래시 및 삭제 시 옵저버 충돌 방지 로직 적용.

## 🛠 주말에 이어서 할 작업 (미완의 과제)

### 1. 쓰레기 필터 미세 조정 (가장 중요)
- **현상:** 큰 폰 테스트 결과, 가족사진은 잘 지켜졌으나 '의미 없는 사물 사진'이 정상(Diet) 목록에 일부 섞여 들어감.
- **할 일:** 정상 목록을 훑어보며 "이건 쓰레기로 가야 하는데" 싶은 사진들의 특징(라벨) 찾기.
- **방법:** `ImageContentClassifier.kt`의 `trashKeywords`에 '사물 쓰레기' 키워드를 아주 조심스럽게 추가 (예: "Wall", "Floor" 등 - 단, 인물 사진에 영향 없는지 확인 필수).

### 2. 1번 폰(사진 많은 폰) 최종 검증
- 981장의 정상 사진 중 '베스트 컷'이 제대로 골라지는지 2~4단계 파이프라인 흐름 관찰.
- 삭제 작업 시 튕김이나 로딩 지연이 완전히 사라졌는지 반복 테스트.

### 3. UI/UX 디테일
- 로고 클릭 시 뜨는 'Bes2 소개' 다이얼로그의 문구나 디자인이 리퍼메타만큼 매끄러운지 확인.
