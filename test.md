# Bes2 내부 베타 테스트 체크리스트

## 1. 초기 설정 및 기본 동작 (Onboarding)

| 구분 | 테스트 항목 | 기대 결과 | 확인 | 비고 |
|---|---|---|---|---|
| **설치/실행** | 앱 최초 실행 | 권한 요청 팝업(저장공간, 알림)이 정상적으로 뜨는가? | | |
| **권한** | 권한 허용 후 진입 | 권한을 모두 허용하면 홈 화면으로 진입하는가? (앱 재시작 없이) | | |
| **로그인** | 구글 포토 로그인 | 설정 > 구글 로그인 버튼 클릭 시 계정 선택창이 뜨고 로그인이 되는가? | | |
| **기본 설정** | 동기화 옵션 확인 | 설정 화면 진입 시 동기화 옵션이 **'분석 직후 바로바로 동기화'**로 선택되어 있는가? | | |
| **스캔** | 초기 스캔 동작 | 홈 화면 진입 시 하단에 '스캔 중/분석 중' 상태가 표시되며 숫자가 올라가는가? | | |

## 2. 핵심 기능 테스트 (Main Features)

### A. 쓰레기통 (스크린샷/불필요 사진)
| 구분 | 테스트 항목 | 기대 결과 | 확인 | 비고 |
|---|---|---|---|---|
| **탐지** | 스크린샷 인식 | 갤러리의 스크린샷 개수와 앱의 '정리할 사진' 개수가 일치하는가? | | |
| **보관** | 선택 사진 보관 | 사진을 선택 후 '보관' 시 목록에서 즉시 사라지고, 홈 화면 숫자도 0으로 줄어드는가? | | |
| **삭제** | 선택 사진 삭제 | 사진을 선택 후 '삭제' 시 실제 갤러리(휴지통)로 이동하고 앱 목록에서 사라지는가? | | |
| **동기화** | 보관 후 동기화 | 보관 처리한 사진이 설정된 옵션(바로바로)에 따라 구글 포토에 업로드되는가? | | |

### B. 갤러리 다이어트 (AI 분류)
| 구분 | 테스트 항목 | 기대 결과 | 확인 | 비고 |
|---|---|---|---|---|
| **진입** | 리뷰 화면 진입 | '갤러리 다이어트' 카드 클릭 시 리뷰 화면으로 넘어가는가? | | |
| **UI** | 베스트/나머지 구분 | 베스트 2장과 나머지/실패 사진이 명확히 구분되어 표시되는가? | | |
| **조작** | 사진 선택/해제 | 베스트가 아닌 사진을 터치하면 베스트로 올라가고, 기존 베스트는 내려오는가? | | |
| **확대** | 사진 확대 보기 | 사진을 길게 누르면 확대 팝업이 뜨고, 좌우 스와이프로 넘겨볼 수 있는가? | | |
| **거부** | 삭제 권한 거부 | '삭제하고 완료' 클릭 -> 시스템 팝업에서 **'거부'** 클릭 시, 다음으로 넘어가지 않고 현재 화면에 머무르는가? | | |
| **완료** | 정리 완료 | '삭제하고 완료' 클릭 -> **'허용'** 클릭 시, 삭제된 사진은 사라지고 다음 묶음으로 넘어가는가? | | |
| **탈출** | 중간 뒤로가기 | 리뷰 도중 좌측 상단 뒤로가기(또는 폰 뒤로가기) 시 홈으로 나가며, **지금까지 정리한 건 동기화**되는가? | | |

### C. 추억 소환 & 리포트
| 구분 | 테스트 항목 | 기대 결과 | 확인 | 비고 |
|---|---|---|---|---|
| **추억** | 추억 카드 표시 | 과거의 특정 날짜(여행 등) 사진이 많을 경우 '추억 소환' 카드가 뜨는가? | | |
| **리포트** | 통계 정확성 | 홈 화면의 오늘 촬영/저장/삭제 카운트가 실제 활동(다이어트/쓰레기 정리)과 일치하는가? | | |
| **리포트** | 상세 리포트 | '상세 리포트' 클릭 시 월간/연간 그래프와 AI 기여도가 표시되는가? | | |

## 3. 인스턴트 분석 (Instant Mode)

| 구분 | 테스트 항목 | 기대 결과 | 확인 | 비고 |
|---|---|---|---|---|
| **촬영** | 연속 촬영 | 카메라 앱으로 사진 5장 정도를 연속으로 찍고 폰을 끈다(또는 홈으로). | | |
| **알림** | 분석 알림 | 잠시 후 상단바에 **'방금 찍은 사진 정리'** 알림이 오는가? | | |
| **진입** | 알림 클릭 | 알림 클릭 시 해당 사진 묶음의 리뷰 화면으로 바로 이동하는가? | | |
| **우선순위** | 백그라운드 분석 | (중요) 기존에 다이어트 분석 중이었더라도, 사진을 찍으면 **인스턴트 알림이 최우선으로** 빨리 오는가? | | |

## 4. 동기화 및 설정 (Sync & Settings)

| 구분 | 테스트 항목 | 기대 결과 | 확인 | 비고 |
|---|---|---|---|---|
| **권한** | 권한 요청 알림 | 동기화가 필요한데 권한이 없을 경우, **'구글 포토 권한 필요'** 알림이 뜨는가? | | |
| **수동** | 수동 동기화 | 설정 > '지금 바로 동기화' 버튼 클릭 시, 대기 중인 사진들이 업로드되는가? | | |
| **옵션** | 와이파이 전용 | 'Wi-Fi에서만 업로드' 켜짐 상태에서 LTE/5G로 변경 시 업로드가 멈추는가(대기하는가)? | | |
| **결과** | 구글 포토 확인 | 실제 구글 포토 앱/웹에 접속했을 때, Bes2에서 **'보관(KEPT)'한 사진만** 올라와 있는가? | | |

## 5. AI 분석 로직 품질 (AI Quality Check)

| 구분 | 테스트 항목 | 기대 결과 | 확인 | 비고 |
|---|---|---|---|---|
| **흔들림** | 흔들린 사진 | 흔들리거나 초점이 나간 사진이 **'실패한 사진'** 그룹으로 잘 분류되는가? | | |
| **눈 감음** | 눈 감은 사진 | 인물 사진 중 눈을 감은 사진이 '눈 감김' 태그와 함께 실패 그룹으로 가는가? | | |
| **베스트** | 베스트 선정 | 연사(비슷한 사진) 중 구도나 표정이 가장 좋은 사진이 '베스트 2장'에 선정되는가? | | |
| **역광** | 역광 보정 | 역광인 사진은 점수가 낮게 책정되거나 적절히 분류되는가? | | |
| **문서** | 문서/사물 | 영수증, 문서, 단순 사물 사진이 '쓰레기통'이나 '나머지'로 잘 분류되는가? | | |
